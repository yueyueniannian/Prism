# Paper Audit Lab

静态前端，直接双击 `index.html` 即可运行。

## 用法

1. 打开 `index.html`
2. 导入 `TXT`、`DOCX` 或 `PDF`
3. 页面会提取正文并允许手动修正
4. 点击“开始评估”
5. 点击右上角“研究图谱”可进入关键词图谱页

## 能力

- 评估可复现性
- 判断是否开源
- 对开源真实性做 GitHub 在线核验
- 判断结果是否合理
- 在原文中按四个维度分颜色高亮
- 证据提取与原文高亮一一对应，可点击跳转
- 新增研究方向图谱页，可搜索方向关键词并展示论文节点关系与方向推荐

## 说明

- `TXT` 可离线导入
- `DOCX`、`PDF` 依赖浏览器联网加载解析脚本
- 旧版 `.doc` 不支持稳定前端解析，建议先转成 `.docx`
- 联网时会自动调用 GitHub Public API 做仓库核验
- 目前优先支持论文正文里的 GitHub 链接
- `research-map.html` 是独立的研究图谱页面
